# USTM LMS — Learning Management System

A role-based Learning Management System built for the University of Science & Technology Meghalaya (USTM).  
This project includes separate portals for **Students**, **Teachers**, and **Admins**, with a modern UI built in React and a backend planned around Spring Boot, MySQL, and REST APIs.

---

## Overview

USTM LMS is designed to handle:

- Student learning and course access
- Teacher course management, assignments, attendance, and announcements
- Admin-level control panels for students, teachers, courses, reports, settings, and analytics
- Role-based navigation and dashboard experience
- Scalable backend integration for university operations

---

## Features

### Student Portal
- USTM landing page
- Course browsing
- Course details
- Profile page
- Learning dashboard
- Assessment/certificate pages

### Teacher Portal
- Teacher dashboard
- My courses
- Assignments management
- Attendance tracking
- Announcement publishing
- Analytics dashboard
- Faculty profile page

### Admin Portal
- Admin dashboard
- Student management
- Teacher management
- Courses management
- Assignment analytics
- Reports dashboard
- System settings
- Role-based sidebar navigation
